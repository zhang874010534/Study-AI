### Jupyter

> Jupyter 是数据科学、机器学习和 Python 开发最流行的交互式开发环境，支持 40+ 编程语言，能将代码、文本、公式和可视化整合在一个文档中。**JupyterLab 是官方推荐的现代界面**，本教程以 JupyterLab 为主，同时兼容传统 Jupyter Notebook。\

#### 一、快速安装

```bash
# 安装 JupyterLab（使用清华镜像加速）
pip install jupyterlab -i https://pypi.tuna.tsinghua.edu.cn/simple

# 验证安装
jupyter lab --version
```
